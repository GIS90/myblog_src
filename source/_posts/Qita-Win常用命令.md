> 方式

command + R打开允许，输入命令。

> 常用命令

|          describe          |       command        |
|:--------------------------:|:--------------------:|
|           控制台           |         cmd          |
| 打开资源管理器（我的电脑） |       explorer       |
|            注销            |        logoff        |
|       关机（60s后）        |       shutdown       |
|         计算机管理         |     compmgmt.msc     |
|          系统服务          |     services.msc     |
|         任务管理器         |       taskmgr        |
|           记事本           |       notepad        |
|          垃圾整理          |       clearmgr       |
|     开启/关闭XXXXX服务     | net start/stop XXXXX |
|           计算器           |         calc         |
|           注册表           |     regedit.exe      |
|        远程桌面连接        |        mstsc         |
